SSA Student Tools Home

This tool provides you downloadable file for your calendar.

If there is anything that also goes through the cycle day that is not a course, it can be entered here.

Instead of typing in your schedule, now you can simply upload the pdf version of it.

More functions coming soon!

Hosting teachers' calendar as subscription to enable more convenient scheduling of meetings.
